用户角色表（User_Role）

| 字段名称 | 类型 | 说明 | isNull |
| -------- | ---- | ---- | ------ |
| userId   | int  | 外键 | no     |
| roleId   | int  | 外键 | no     |

数据库sql

```mysql
create table user_role(
    userId int not null,
    roleId int not null,
    primary key(userId, roleId),
    constraint fk_user_id foreign key(userId) references user(id),
    constraint fk_role_id foreign key(roleId) references role(id)
);
```
